## Scaffolding Your First Popoyoko Project :

npm :

```bash
npm create popoyoko
```

bun :

```bash
bun create popoyoko
```
OR

bunx :

```bash
bunx create-popoyoko
```

Uses the Popoyoko-UI-Token-Generator to create your tokens
