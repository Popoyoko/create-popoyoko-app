## Scaffolding Your First Popoyoko Project :

npm :

```bash
npm create popoyoko
```

bun :

```bash
bun create popoyoko
```
OR

bunx :

```bash
bunx create-popoyoko
```

Uses the create-popoyoko-variables to create your tokens
